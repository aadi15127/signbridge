# SignBridge

Real-time sign language translator for video calls using MediaPipe and TensorFlow.js. Detects ASL gestures with micro-gestures and emotional context.

## Features
- Real-time hand gesture recognition
- Facial emotion analysis
- Micro-gesture detection
- Live text overlays
- Demo mode
- Session logging

## Tech Stack
- MediaPipe Hands & FaceMesh
- WebRTC
- Vanilla JS with modular architecture
- Tailwind-inspired CSS

## Setup
1. Clone the repo
2. Open `index.html` in browser
3. Allow camera access
